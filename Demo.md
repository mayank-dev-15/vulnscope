## About

**vulnscope** is a vulnerability intelligence platform.

### Key Features

- Real-time CVE monitoring via NVD API
- Exploit correlation (CIRCL)
- Risk scoring engine (0-10 scale)
- Beautiful React dashboard
- REST API with FastAPI
- Async data fetching
- Search and filter CVEs
- Trend analysis and statistics

---

## Tech Stack

- Python
- FastAPI
- React
- NVD API
- CIRCL API
- SQLite
- Docker

---

## Controls & Usage

| Control | Action |
|---------|--------|
| `Nav bar` | Switch views |
| `Search` | Find CVEs |
| `Filters` | Filter by severity/date |
| `Click CVE` | View details |
| `Export` | Download report |
